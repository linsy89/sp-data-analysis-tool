# 属性提取规则修正说明

## 修正背景

经过验证，原有的属性提取规则需要调整，以满足实际业务需求。

---

## 新旧规则对比

### ❌ 旧规则（错误）
**描述**：第一个空格之后的**所有内容**

**示例**：
```
City-LightballsCollage-IP15 aesthetic
                          ↑
                    第一个空格
                          → "aesthetic" ✓

City-BlueArt-IP16P collage ip16p
                  ↑
            第一个空格
                  → "collage ip16p" ✗（不应该包含"ip16p"）

Flower-FloralSky-IP15 人群 ip15
                     ↑
               第一个空格
                     → "人群 ip15" ✗（不应该包含"ip15"）
```

---

### ✅ 新规则（正确）
**描述**：第一个空格之后的**第一个单词**（只到第二个空格前）

**示例**：
```
City-LightballsCollage-IP15 aesthetic
                          ↑
                    第一个空格
                          → "aesthetic" ✓

City-BlueArt-IP16P collage ip16p
                  ↑
            第一个空格
                  → "collage" ✓（只取第一个单词）

Flower-FloralSky-IP15 人群 ip15
                     ↑
               第一个空格
                     → "人群" ✓（只取第一个单词）
```

## 代码修改

### data_processor.py - extract_dimensions()

#### 修改前：
```python
# 提取属性（第一个空格之后的内容）
if ' ' in campaign_name:
    attribute = campaign_name.split(' ', 1)[1].strip()
else:
    attribute = '未分类'
```

#### 修改后：
```python
# 提取属性（第一个空格之后的第一个单词）
if ' ' in campaign_name:
    # 分割所有单词
    words = campaign_name.split(' ')
    # 取第一个空格之后的第一个单词（即index=1）
    attribute = words[1].strip() if len(words) > 1 else '未分类'
else:
    attribute = '未分类'
```
